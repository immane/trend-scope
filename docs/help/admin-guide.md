# Trend-Scope 管理后台使用指南

## 导航概述

管理后台地址：`http://localhost:3000`

侧边栏菜单包含以下功能模块：

| 菜单 | 功能 |
|---|---|
| Dashboard | 全局统计、调度状态、最近回测 |
| 标的管理 | 查看/新增/编辑/删除标的，同步行情数据 |
| 策略管理 | 创建/编辑策略，11 种内置模板或自定义 Python 脚本 |
| 回测历史 | 查看所有回测记录，多选对比，查看详情 |
| 信号 | 查看触发信号，生成 AI 分析 |
| 提醒日志 | 查看已发送的邮件提醒 |
| 提醒规则 | 管理用户的提醒规则（启停、筛选） |
| 用户管理 | 管理用户账号、角色、状态 |
| 内容管理 | 管理公告通知 |
| 数据管理 | 查看每个标的的行情数据概况、同步/清除数据 |
| AI 设置 | 配置 AI 分析服务的 API 密钥和模型 |

## Dashboard

展示 6 个统计卡片：用户总数、标的总数、策略数量、信号数量、提醒次数、回测总数。下方显示调度服务状态和最近回测记录。

## 标的管理

- **列表页**：Symbol、名称、类型、行业、现价和涨跌、15 日走势图、状态、最后更新日期
- **详情页**：富报价（开盘/最高/最低/昨收/成交量/数据量）、K 线图（蜡烛+MA+成交量+MACD）、52 周区间、各周期收益
- 列表页可同步全部行情或单个标的

## 策略管理

- **创建策略**：从 11 种模板中选择，自动填充 Python 代码，可在 Monaco 编辑器中修改
- **策略详情**：编辑代码和参数、运行回测、查看历史回测记录
- 策略定义了完整的交易逻辑，通过 Python 的 `analyze(df, params)` 函数返回买卖信号
- 详细协议参考 `docs/help/strategy-script.md`

## 回测历史

- **列表页**：按策略收益率排序，支持多选对比
- **详情页**：权益曲线、回撤曲线、月度收益分布、交易明细、完整分析指标
- **对比页**：多策略归一化收益曲线叠加和指标对比表

## 信号

- 列表显示所有触发的买卖信号，可按类型筛选
- 点击信号行查看详情，包含信号基础信息、触发详情 JSON 和 AI 分析报告
- 生成 AI 分析：点击"生成分析"按钮，系统会调用配置的 AI 模型生成结构化分析

## AI 设置

- 查看/修改 AI 配置：API Key、Base URL、模型名称、运行时开关
- 运行时配置覆盖环境变量，即时生效，无需重启
- 支持任何 OpenAI 兼容的 API（DeepSeek、OpenAI、自托管模型等）
- 配置参考：`docs/help/ai-config.md`

## 提醒日志

- 查看所有已发送的提醒邮件
- 点击行查看邮件主题和完整邮件内容

## 提醒规则

- 查看所有用户的提醒规则
- 可按用户 ID 和状态筛选
- 支持启停单个规则

## 用户管理

- 查看所有用户，按角色和状态筛选
- 提升管理员 / 封禁 / 解禁用户
- 点击用户行查看详情、修改基本资料、查看最近提醒和回测

## 内容管理

- 创建/编辑/删除公告
- 发布/取消发布、置顶/取消置顶

## 数据管理

- 每个标的的数据行数、最早/最近日期、数据来源
- 同步单个标的最新行情
- 清除某个标的全部历史价格数据
